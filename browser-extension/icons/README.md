# Placeholder for browser extension icons

To create actual icons, you can:

1. **Use an online icon generator**:
   - Go to https://www.favicon-generator.org/
   - Upload a logo or design
   - Generate 16x16, 48x48, and 128x128 PNG files

2. **Create manually**:
   - Use any image editor (Photoshop, GIMP, Figma)
   - Create images with these dimensions:
     - icon16.png (16x16 pixels)
     - icon48.png (48x48 pixels)
     - icon128.png (128x128 pixels)

3. **Use placeholder**:
   For now, the extension will work without icons (browser will show default icon)
